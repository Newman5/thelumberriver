# 1. Overview

thelumberriver.com is a website focused on the natural beauty, environmental science, and ecology of the Lumber River in Robeson County, NC.

Tone: clear, concise, accessible (8th‑grade reading level), with moments of gentle poetry.

Target Audience: visitors to the Lumber River — birders, paddlers, hikers, fishermen, nature lovers.

Main Tasks for Copilot:

Generate page content and layout

Create components

Implement Tailwind CSS styling

Generate placeholder images

Produce documentation

Create SEO‑friendly metadata

Suggest improvements when possible

This site should attract search‑engine traffic and gradually expand to include sponsored posts for local businesses (e.g., fishing guides, outfitters, paddling rentals). Early emphasis is on creative nature content such as flora & fauna posts.

# 2. Requirements

Framework: Astro

Deployment: Cloudflare Pages

Styling: Tailwind CSS

Responsive: Mobile‑first

SEO: metadata, schema.org, sitemap, robots.txt

Images: optimized, lazy‑loaded, descriptive alt text

Content Strategy: Copilot‑generated blog posts about wildlife, paddling guides, and seasonal updates

Coding Style: use only basic, beginner‑level techniques found in the Astro.build tutorial
